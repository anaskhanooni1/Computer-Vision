def ann_model_plant_seeds_class(neurons_count, output_neurons_count, 
                  initializer, act_function, act_function_output_layer,
                  learning_rate, no_epochs, no_batches, add_batchnormalization_layer, 
                  add_dropout_layer, dropout_percentage,
                  print_model_summary):
  
  #Creating a model
  nn_model = Sequential();

  # INPUT Layer
  nn_model.add(Dense(neurons_count, input_shape = (12288,), kernel_initializer = "he_normal"))
  nn_model.add(Activation(act_function))

  # HIDDEN Layer - 1
  nn_model.add(Dense(neurons_count, kernel_initializer = "he_normal"))
  nn_model.add(Activation(act_function))

  # OUTPUT Layer
  nn_model.add(Dense(output_neurons_count, kernel_initializer = "he_normal"))
  nn_model.add(Activation(act_function_output_layer))

  #Adding the compilation layer
  sgd = optimizers.SGD(learning_rate)
  nn_model.compile(optimizer = sgd, loss = 'categorical_crossentropy', metrics = ['accuracy'])
  
  #Print the model summary
  if print_model_summary:
    nn_model.summary()

  nn_model.fit_generator(training_image_set, steps_per_epoch = int(4767/32), epochs = 20, verbose = 1)
  print(nn_model.history.history.keys())
  print("nn_model.history.history")
  return nn_model

Task 2

Among the there above model, CNN turns out to be the most effiecient model. In Convolutional neural network, the kernels used for pixel wise feature extraction from image. The use of forward and backward propagation is also very useful in reducing error after each epoch. After each epoch, the neurons and the weights are adjusted accordingly for reducing the epoch error and increasing overall model accuracy. In CNN, the dense layers process the input from previous layer and gives more refined output to the next layer. CNN supports multiscale image processing upto more extent as compared to supervised learning algoritms. The feature extraction is followed by classification with fully connected layer increases the model accuracy.
